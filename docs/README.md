## Project Overview

**iComplain** is a web-based facility management system designed to streamline the reporting, tracking, and resolution of facility issues within the USLS - Bacolod campus. The platform provides students with an easy way to report facility problems while giving administrators powerful tools to manage and resolve issues efficiently.
